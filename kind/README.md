  kind: tool for running local Kubernetes clusters using Docker container "nodes".

To update or switch versions, run `webi kind@stable` (or `@v2`, `@beta`,etc).

## Cheat Sheet

To create a cluster 
```bash
kind create cluster
```

To delete a cluster 
```bash
kind delete cluster
```
